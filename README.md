# 빕스 VIPS 리뉴얼
기존 빕스 사이트의 낙후함에 따라 와이드하고 트렌디하게 리디자인 해보았습니다. 스크롤할 때, 한 단락씩 화면이 변경되도록 플러그인 없이 제이쿼리로 구현해보았습니다.
CSS를 통해 반응형으로 퍼블리싱하였습니다.
